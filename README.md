# practical-test-for-fullstack-javascript-developer

## Project setup
```
Clone the repo
Run npm install
Create a .env file 
In the .env file you will need the following in other to run the project
VUE_APP_URI
PORT 
DATABASE_LOCAL 
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
